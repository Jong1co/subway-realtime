# 위치 기반 실시간 지하철 앱

현재 위치 기반으로 주변의 실시간 접근 지하철을 확인할 수 있는 어플입니다.

**기능**

- 실시간 위치 기반 주변 역 탐색
- 지하철 역 검색 기능
- 지하철 즐겨찾기 기능

**기술 스택:** React Native, React Query, Recoil, Async Storage

**API:** 서울열린데이터광장
